## Resolved (Phases 1-5)

- Branch relaxation wired into pipeline (short/near form selection)
- Sext/Zext/Trunc machine lowering (X86Movsx/Movzx/Trunc)
- Load/Store effectful op lowering to MovRM/MovMR
- Coalescing wired with real block-param copy pairs
- Loop-depth aware spill selection
- RCX pre-coloring for shift count operands
- XMM spill/reload markers
- RPO block ordering with fallthrough optimization
- Call result handling (CallResult op, RAX pre-coloring)
- Phi elimination stress-tested (4-way cycles, diamond CFG)

## Remaining Known Limitations

### Pipeline
- Caller-saved registers not marked as clobbered across Call instructions
  (values live across calls in caller-saved regs may be corrupted)
- Cross-block live range splitting not implemented (single flat regalloc pass;
  long-lived values increase register pressure across blocks)

### Code Quality (Future Features)
- No SIMD / AVX
- No x87 FP
- No debug info (DWARF)
- No unwind tables
- No .note.GNU-stack section (linker warns about executable stack)
